# ultima-lectio
a facebook clone make using mern stack for the odin project.

a link to the website: https://luxury-lebkuchen-233be4.netlify.app/

features

     -likes
     -comments
     -adding friends
     -saving posts
     -sharing posts
     -tags
     -post searching
     -notifications
     -re-sizes to fit on phone screens
     -google login
